Browser Not Currently Supported

Unfortunately, DrumKid doesn't work on this browser. DrumKid requires a fairly new feature called Web Audio, which is not available on all browsers. You may be able to run DrumKid in a different or updated browser; the following browsers are known to work:

  • Google Chrome on Windows, Mac and Linux (recommended)
  • Safari on iOS (recommended)
  • Google Chrome Beta on Android and iOS (possibly)
  • Firefox on Windows, Mac and Linux (possibly, on the latest version)
  • Opera on Windows, Mac and Linux (possibly)
